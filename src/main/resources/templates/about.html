О программе

Данное приложение предназначено для управления строительной компанией. В нем реализован удобный интерфейс для работы с основными сущностями компании:

  • Стройки — управление объектами строительства и этапами работ.
  • Бригады и сотрудники — учет рабочих бригад и их сотрудников с возможностью добавления, редактирования и удаления. Подробнее см. в разделе Сотрудники бригад.
  • Клиенты — хранение контактной информации и данных клиентов компании. Позволяет создавать, редактировать и удалять клиентов. Подробнее в разделе Список клиентов.
  • Дефекты, материалы, поставщики, стадии — инструменты для отслеживания качества и управления ресурсами.
  • Администрирование пользователей — управление учетными записями и ролями (только для администраторов).

Основные функции приложения

  • Поиск по имени в списках сотрудников и клиентов для быстрого доступа к нужным данным.
  • Добавление новых записей через удобные модальные окна без перезагрузки страницы.
  • Редактирование и удаление существующих данных с подтверждением действий.
  • Роли и права доступа, обеспечивающие безопасность и разграничение функционала.
  • Навигация по разделам через нижнее меню, доступное на всех страницах.

Навигация

Для работы с сотрудниками бригад перейдите в раздел Сотрудники бригад. Здесь доступен поиск по имени, просмотр полного списка, а также операции добавления и редактирования с помощью модальных окон.

Для управления клиентами используйте раздел Клиенты. В нем реализованы аналогичные возможности с таблицей, поиском и модальными формами для создания и редактирования клиентов.

Технологии

Приложение построено на Spring Boot с использованием Thymeleaf для серверного рендеринга страниц и Bootstrap 5 для адаптивного и современного интерфейса.
