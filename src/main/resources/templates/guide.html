Руководство пользователя

О программе
Как пользоваться системой

Добро пожаловать в информационную систему управления строительной компанией. На этой странице представлено руководство пользователя, которое поможет вам эффективно использовать возможности системы.

1. Введение

Данная система предназначена для управления строительными объектами, дефектами, бригадами, рабочими, клиентами, материалами, поставщиками, стадиями строительства и пользователями. Это руководство поможет вам освоить основные функции приложения.

2. Авторизация и регистрация
2.1. Вход в систему

  • Откройте страницу авторизации (/login).
  • Введите Логин и Пароль в соответствующие поля.
  • (Опционально) Установите флажок Запомнить меня для сохранения сессии.
  • Нажмите Авторизоваться.
  • При неверных данных появится сообщение об ошибке. Проверьте логин и пароль.

2.2. Регистрация

  • На странице авторизации нажмите Еще нет аккаунта? или перейдите на /registration.
  • Заполните поля:
    • Логин: Уникальное имя пользователя (например, лимон1724). Должно быть уникальным в системе.
    • Пароль: Надежный пароль.
    • Почта: Действующий email (например, user@example.com). Должен быть уникальным.
    • Номер телефона: В формате +7(000)000-00-00 (опционально).
  • Нажмите Зарегистрироваться.
  • Если логин или email уже заняты, появится сообщение об ошибке. Выберите другой логин или email.
  • Для отмены нажмите Вернуться.

2.3. Выход из системы

  • В нижней части любой страницы найдите кнопку Выход (красная кнопка в футере).
  • Нажмите Выход, чтобы завершить сеанс.

3. Навигация

  • Используйте нижнее меню для перехода между разделами: Стройки, Клиенты, Бригады, Рабочие, Дефекты, Материалы, Поставщики, Стадии, Пользователи (последний доступен только для ADMIN).
  • Разделы Клиенты, Бригады, Рабочие, Дефекты, Материалы, Поставщики, Стадии и Пользователи доступны только пользователям с ролью ADMIN.
  • Нажмите Руководство пользователя на любой странице, чтобы открыть это руководство (/guide).

4. Раздел «Строительные объекты»

Страница предназначена для управления строительными объектами. Пользователи могут просматривать список объектов, добавлять новые, редактировать или удалять существующие, а также просматривать детали объекта.

4.1. Поиск по названию объекта

  • В верхней части страницы введите название (или часть названия) в поле Поиск по названию.
  • Нажмите Поиск.
  • Отобразится список объектов, соответствующих запросу.

4.2. Просмотр списка объектов

  • Перейдите в раздел Стройки (/constructionSites).
  • В таблице отображаются:
    • ID: Уникальный идентификатор объекта.
    • Название: Название объекта.
    • Локация: Место строительства.
    • Начало: Дата начала проекта.
    • Окончание: Планируемая дата завершения.
    • Бюджет: Бюджет проекта.
    • Действия: Кнопки для просмотра деталей, редактирования или удаления (для ADMIN).

4.3. Добавление нового объекта

  • Нажмите Добавить объект (доступно для ADMIN).
  • В модальном окне заполните:
    • Название: Название объекта.
    • Локация: Место строительства.
    • Дата начала: Дата начала проекта.
    • Дата окончания: Планируемая дата завершения.
    • Бюджет: Бюджет в числовом формате.
  • Нажмите Сохранить.
  • Для отмены нажмите Отмена.

4.4. Редактирование объекта

  • В таблице нажмите Редактировать напротив нужного объекта (доступно для ADMIN).
  • В модальном окне измените необходимые поля.
  • Нажмите Сохранить.
  • Для отмены нажмите Отмена.

4.5. Удаление объекта

  • В таблице нажмите Удалить напротив объекта (доступно для ADMIN).
  • Подтвердите удаление в появившемся окне.
  • Объект будет удален.

4.6. Просмотр деталей объекта

  • В таблице нажмите Детали напротив объекта.
  • На странице отобразятся:
    • Название, Локация, Дата начала, Дата окончания, Бюджет.
    • Список Дефектов, связанных с объектом.
    • Список Бригад, назначенных на объект.
    • Список Стадий строительства.
  • Нажмите Назад к списку, чтобы вернуться.

5. Раздел «Дефекты»

Страница предназначена для управления дефектами на строительных объектах. Пользователи могут просматривать, искать, добавлять, редактировать и удалять дефекты.

5.1. Поиск по описанию дефекта

  • В верхней части страницы введите часть описания в поле Поиск по описанию.
  • Нажмите Поиск.
  • Отобразится список дефектов, соответствующих запросу.

5.2. Просмотр списка дефектов

  • Перейдите в раздел Дефекты (/defects, доступен для ADMIN).
  • В таблице отображаются (по 10 записей на страницу):
    • ID: Уникальный идентификатор дефекта.
    • Описание: Описание проблемы.
    • Статус: Текущий статус (например, "Открыт").
    • Дата: Дата регистрации.
    • Объект: Название связанного объекта или "Не привязан".
    • Действия: Кнопки для редактирования или удаления.

5.3. Добавление нового дефекта

  • Нажмите Добавить дефект.
  • В модальном окне заполните:
    • Описание: Подробное описание дефекта.
    • Статус: Текущий статус.
    • Дата: Дата регистрации.
    • Объект: Выберите строительный объект из списка или оставьте "Не привязан".
  • Нажмите Создать.
  • Для отмены нажмите Отмена.

5.4. Редактирование дефекта

  • В таблице нажмите Редактировать напротив дефекта.
  • В модальном окне измените поля, включая возможность сменить объект.
  • Нажмите Сохранить.
  • Для отмены нажмите Отмена.

5.5. Удаление дефекта

  • В таблице нажмите Удалить напротив дефекта.
  • Дефект будет удален без подтверждения (будьте внимательны).

5.6. Навигация по страницам

  • Используйте пагинацию внизу таблицы (10 записей на страницу).
  • Нажмите Назад, Вперёд или номер страницы для перехода.

12.2. Просмотр списка пользователей

  • Перейдите в раздел Пользователи (/admin/user/list, доступен для ADMIN).
  • В таблице отображаются пользователи с их данными:
    • ID: Уникальный идентификатор пользователя.
    • Имя: Логин пользователя.
    • Email: Электронная почта.
    • Телефон: Контактный номер.
    • Роль: Роль пользователя (например, USER или ADMIN).
    • Действия: Кнопки для редактирования или удаления.

12.3. Редактирование пользователя

  • Нажмите Редактировать напротив пользователя.
  • На странице редактирования измените:
    • Имя: Логин пользователя.
    • Email: Электронная почта.
    • Телефон: Контактный номер.
    • Пароль: (Опционально) Новый пароль.
    • Роль: Выберите роль (USER или ADMIN).
  • Нажмите Сохранить.
  • Для отмены вернитесь назад через браузер.

12.4. Удаление пользователя

  • Нажмите Удалить напротив пользователя.
  • Пользователь будет удален без подтверждения (будьте внимательны).

13. Советы по использованию

  • Убедитесь, что строительные объекты, бригады и поставщики созданы перед добавлением дефектов, рабочих, материалов или стадий.
  • Используйте поиск для быстрого нахождения записей.
  • Будьте внимательны при удалении записей, так как многие действия (например, удаление бригад, рабочих, дефектов, материалов, стадий, пользователей) не требуют подтверждения.
  • Проверяйте корректность дат, числовых значений (например, бюджет, количество материалов) и уникальность данных (например, логин и email при регистрации) при добавлении или редактировании.
